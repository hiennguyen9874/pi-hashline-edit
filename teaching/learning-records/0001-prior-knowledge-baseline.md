# Prior knowledge baseline established

The learner has used `read`, `edit`, and `insert` as an agent user and understands their purpose and interface, but has never read the source code. They want to learn the hashline protocol, tool architecture, and edit pipeline deeply.

**Implications:** Lessons should start at the protocol level (hash computation, anchor resolution) before moving to tool registration and the edit pipeline. No need to explain what the tools do at the user level — the learner already knows that from experience.
